# SpringBoot-Security-JWT-React
Security operation is used in React and Spring Boot (with Redux and without Redux)

## Using Tools & Technologies
### Backend
``` 
- Spring Boot 2
- Spring Security, JWT
- REST API (get, post, put, delete, patch)
- Lombok
- ModelMapper, DTO
- JPA, Hibernate
- PostgreSql 12
```
### Fackend
``` 
- React 8
- Redux
- Alertify Js
- Bootstrap 4,
```
